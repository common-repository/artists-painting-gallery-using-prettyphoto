== Description ==

Gallery using PrettyPhoto, for use with Artwork when you would like to add the date, medium, dimensions and associated with a particular page in the Media Gallery.

This plugin is maintained by Michael MacMillan, http://withsquirrelly.com or you can find him on Twitter at http://twitter.com/withSquirrelly/

== Installation ==

1. Upload the files under `/artists-painting-gallery-using-prettyphoto/` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
